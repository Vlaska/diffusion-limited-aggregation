# Diffusion Limited Aggregation


## Requirements

- Python >=3.8
- Poetry (`pip install poetry`)
- `gcc` for Linux, `XCode` for MacOS, `Visual Studio` for Windows


## How to run

1. Install `Poetry`
2. `poetry shell`
3. `poetry install --no-dev`
4. `python setup.py develop`
5. `python ./DLA` or `python -m DLA`
